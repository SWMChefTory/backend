== 공통 응답 구조

=== 성공 응답
[source,json]
----
{

}
----
* `userStatus`: HTTP 상태 코드
* `data`: 실제 응답 데이터

=== 실패 응답
[source,json]
----
{

"errorCode": "GLOBAL_1",
"message": "요청 변수가 잘못되었습니다."

}
----
* `errorCode`: 에러 코드
* `message`: 에러 메시지

== 상태 코드

|===
| 상태 코드 | 의미 | 설명
| 200       | OK   | 요청이 성공적으로 처리됨
| 400       | BadRequest | 잘못된 요청
| 500       | Internal Server Error | 서버 내부 오류
|===

=== Authorization
* `Authorization`: Bearer 액세스 토큰 (필수)
* 형식: `Bearer {token}`
* 인증이 필요한 API 호출 시 반드시 포함
